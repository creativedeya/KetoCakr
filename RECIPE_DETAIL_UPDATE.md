# Recipe Detail Screen Update - Installation Instructions

## Какво е направено:

Създадох напълно нов дизайн на екрана за детайли на рецептата (`/Mobile/app/user-recipe/[id].tsx`) с:

### ✨ Нови функции:

1. **Hero изображение с Glassmorphism ефекти**
   - Красиво hero изображение на торта
   - Glassmorphism карти с BlurView ефект
   - Gradient overlay за по-добра четимост

2. **Два режима на показване:**
   - **Режим "Порции"** - променяш броя порции, общото тегло остава същото
   - **Режим "Количество"** - умножаваш/делиш всичко (x0.5, x1, x1.5, x2, x5, etc.)

3. **Динамични изчисления:**
   - Хранителни стойности на порция (автоматично преизчисляване)
   - Общо тегло и тегло на порция
   - Цена (готова за бъдеща имплементация)

4. **Три таба със съдържание:**
   - **Съставки** - списък на продуктите с количества
   - **Стъпки** - инструкции за приготвяне по компоненти
   - **Сглобяване** - (предвидено за бъдещо добавяне)

5. **Допълнителни функции:**
   - Бутон "Добави към списъка за пазаруване"
   - Любими (Heart button)
   - Споделяне (Share button)
   - Обратно навигация

6. **Реални данни от Supabase:**
   - Използва данните от `user_recipes`, `base_recipes`, `dessert_types`
   - Изчислява хранителните стойности от базовите рецепти
   - Показва реалните съставки и инструкции

## 📦 Какво трябва да инсталираш:

Отвори терминал в папка `Mobile` и изпълни:

```bash
npm install expo-blur@~13.0.2
```

## 🚀 Как да тестваш:

1. Инсталирай `expo-blur` (виж по-горе)
2. Стартирай приложението:
   ```bash
   npm start
   ```
3. Отвори някоя рецепта от списъка
4. Тествай:
   - Превключване между "Порции" и "Количество"
   - Промяна на броя порции с + и -
   - Превключване между табовете
   - Бутоните за любими и споделяне

## 🎨 Визуални подобрения:

- **Цветова схема:**
  - Primary: #4169E1 (Royal Blue)
  - Accent: #50E3C2 (Turquoise за нетни въглехидрати)
  - Success: #4CAF50 (Green за shopping list button)

- **BlurView ефекти:**
  - Хранителни стойности: `intensity={80}` с `tint="light"`
  - Контроли: `intensity={60}` с `tint="dark"`

- **Градиенти:**
  - Hero overlay: прозрачен → тъмен (40% → 100%)
