## 🐛 Наткнати проблеми

- **pdfjs word-per-line** — всяка дума на отделен ред. Решение: групиране по Y-координата ±4px
- **413 Too Large** — 35MB PDF → base64 ~47MB надвишава Claude API лимит. Решение: текст екстракция първо
- **JSON truncation** — Claude съкращава отговора в средата на array. Решение: max_tokens 16000 + matchAll fallback за частични обекти
- **Fraction quantities** — "1/4", "1 1/2" отхвърлени от numeric DB колона. Решение: parseQuantity() helper
- **IngredientAutocomplete props изчезнали** — quantity/unit/onRemove props бяха изтрити. Решение: добавени обратно като optional props
